# SwarmCommand

This placeholder directory defines the X3 Tauri console SwarmCommand app.

It should eventually include UI wiring for:
- feature mode (dev/prod toggle)
- backend health (uptime/status)
- last test status (pass/fail/timestamp)
- latest chain event (type/timestamp)
- readiness score (composite metric)
- blockers (severity/categories)

Any disabled buttons should render a clear reason instead of appearing inert.
