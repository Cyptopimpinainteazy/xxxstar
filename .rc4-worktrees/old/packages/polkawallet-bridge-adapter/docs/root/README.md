# @x3-chain/polkawallet-bridge-adapter

Bridge adapter package for Polkawallet integration on X3 Chain.

## Install

```bash
npm install @x3-chain/polkawallet-bridge-adapter
```

## Usage

```ts
import { PolkawalletBridgeAdapter } from '@x3-chain/polkawallet-bridge-adapter';

// Configure adapter and bridge SDK integration per deployment environment.
```
