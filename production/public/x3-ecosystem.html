Four phases.

From secret generation to profit collection. Every step is trustless.

01 / SETUP
Generate the Secret

Compute secret S and hash H=sha256(S). Set staggered timelocks on both chains. Pre-fund inventory vaults with collateral.

S = random_bytes(32)
H = sha256(S)
T_S = now + 3600
T_E = now + 7200
02 / LOCK
Deploy HTLCs

Create hash-locked contracts on both chains with the same H. Wait for confirmations. Funds are locked but not moved.

svm.createHTLC(H, T_S)
evm.createHTLC(H, T_E)
await confirmations
03 / FLASH
Execute Flash Legs

Both chains fire their flashloan→swap→repay sequence. One transaction each. If either reverts, no secret is released.

kamino.flashBorrow(1M)
jupiter.swap(USDC→SOL)
kamino.flashRepay(1M)
04 / CLAIM
Reveal and Collect

Both legs succeed. Initiator reveals S on Solana. Relayer claims the Ethereum side. Profit secured on both chains.

svm.claimHTLC(id, S)
// emit HtlcClaimed(S)
evm.claimHTLC(id, S)
