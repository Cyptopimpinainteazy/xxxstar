# Cross-Chain Governance

- All X3 and wrapped tokens (wX3, xX3, etc.) grant voting power.
- Voting power = sum of native + all wrapped balances.
- Proposals can:
  - Adjust fees
  - Change treasury splits
  - Approve upgrades
  - Distribute rewards
- Voting is open for a set duration (blocks).
- Execution requires majority of total voting power.
- Governance contract is upgradable by DAO proposal.
- AI agents can suggest proposals and reward distributions.
