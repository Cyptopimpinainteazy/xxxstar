# @x3-chain/x3-wallet

Polkawallet JS API plugin for X3 Chain runtime modules.

## Install

```bash
npm install @x3-chain/x3-wallet
```

## Usage

```ts
import * as X3Wallet from '@x3-chain/x3-wallet';

// Use exported plugin/service modules for x3vm, settlement, governance and trades.
```

Build output includes CJS/ESM bundles and typed declarations in `dist/`.
