# X3 Level 10 Agent Roster

## Commander

Owns task order, escalation, stop/go decisions, and one-task-at-a-time patch discipline.

## Scanner

Owns file coverage, inventories, old/current repo comparison, and unreadable-file tracking.

## Cartographer

Owns `.x3/X3_SYSTEM_MAP.md`, `.x3/graph/*`, dependency maps, and GraphOps evidence.

## Integrator

Owns scoped P0/P1 feature migration and implementation.

## Fixer

Owns failing builds, tests, imports, and broken wiring. No new features.

## Auditor

Owns mainnet blockers, launch gates, and proof-vs-claim review.

## Security

Owns bridge, VM, DEX, nonce, replay, randomness, auth, and invariant risks.

## Architect

Owns better design ideas, competitive advantage, and mainnet simplification paths.

## Bench

Owns TPS, proof, benchmark, stress, and performance tooling.

## LaunchOps

Owns chain spec, genesis, validator bootstrap, runtime upgrade rehearsal, and deployment readiness.

## Scribe

Owns `PATCH_LOG.md`, reports, docs, decisions, and status summaries.
